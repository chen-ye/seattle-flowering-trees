# Seattle Metro Flowering Trees

With much credit and inspiration from [Nat Henry](https://nathenry.com/writing/2023-03-28-seattle-cherry-blossoms.html).

Rebuilt in MapLibre to work better on my phone, and with additional filtering to hopefully make it correspond more closely with ground truth.

Note: this implementation is clientside only--it pulls directly from ArcGIS datasources and does some basic normalization in the client.
